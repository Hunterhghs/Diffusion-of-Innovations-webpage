# Diffusion of Innovations — Development Economics

An informational webpage on the Diffusion of Innovations theory in the context of Development Economics.

## Contents

- **Overview** — Foundational theory by Everett M. Rogers
- **Core Elements** — Innovation, adopters, communication channels, time, social system
- **Adopter Categories** — Innovators through laggards; adoption curve
- **Innovation-Decision Process** — Knowledge, persuasion, decision, implementation, confirmation
- **Development Context** — Productivity gaps, agricultural extension, health & sanitation, critiques
